EXPLAINATION:
I am a software engineer who works for Global Rain, a company that specializes in custom software design and development. The Image below is the Grazioso Salvare Rescue Shelter Website application that was created by my team and me. Designed for the Austin animal shelter center to identify good candidate dogs and cats for search-and-rescue training. My role centers on creating secure, efficient, and tailored applications that solve specific client problems, such as replacing paper-based processes or enhancing data protection. The functions in the project include Interactive filters for the data table, along with a geolocation and a pie chart. 

Writing maintainable, readable, and adaptable programs involves modular design, clear commenting, and consistent naming conventions, as demonstrated by creating a reusable CRUD Python module for MongoDB. This approach allows for scalability and easy debugging across projects. As a computer scientist, I approach problems by breaking them into manageable components—initially analyzing data requirements before building functional, interactive, and efficient solutions tailored to client needs.
Computer scientists translate complex problems into efficient, functional code, utilizing database management, algorithms, and system design to optimize workflows.
This work helps companies like Grazioso Salvare by turning raw data into actionable insights, such as identifying the best dogs for specific, high-stakes tasks, ultimately saving time and improving safety.
